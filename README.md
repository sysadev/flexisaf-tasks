# FIP Weekly Deliverables

## Overview
This repository contains a collection of Java programming assignments and exercises, organized by week. Each folder contains the source code and specific instructions for that week's tasks.

## How to Run the Code
To run any assignment, you will need to navigate into the specific folder first.

1. Open your terminal or command prompt.
2. Navigate to the desired week. For example:
   ```
   cd week1
   ```
3. Compile the Java file:
   ```
   javac GenericCounter.java
   ```
4. Run the program:
   ```
   java GenericCounter
   ```
